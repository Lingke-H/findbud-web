# 1		数学建模大赛队友

## 1.1		固定标签

> **固定标签**是用户的硬性分类依据，用于确定待选其他用户群体的最小子集，作用方式是条件判断型筛选

1. 对搭子的性别要求
2. 对搭子的年级要求
3. **目标（想要找什么类型的搭子，在这里就是数学建模大赛队友）**
4. 是否想要长期组队

## 1.2		分类标签

> **分类标签**用于在满足固定标签条件的其他用户集合中的精筛，作用方式是效用函数取值大小型筛选

### 1.2.1		互斥标签

> **互斥标签**是指在某一类别（组）的标签下，所有子标签大小的和是不变的，通俗来说，一个标签的指标高，另一个标签的指标就会低（所谓”此消彼长“）。一般情况下，互斥标签意味着互补，例如某用户在建模手的维度评分较高（适合担任建模手），那么他就更需要在编程手或论文手维度上评分高的搭子

#### 标签类别_1：技能向量

> 综合考虑用户的**主观意愿**和**主观能力评估**。在相同维度（比如说“建模手”）上同为评分高的两个用户**更不可能**被匹配为搭子

1. 建模手
2. 编程手
3. 论文手

#### 标签类别_2：性格动能因子

> 考虑用户的性格特质。在相同维度上同为评分高的两个用户**更不可能**被匹配为搭子

1. 军官（领导者）
2. 勇士（执行者）
3. 军师（支持者）

### 1.2.2		独立标签

> **独立标签**反映一个用户相对独立的指标（和其他标签没关系，不存在互斥标签中”此消彼长“的关系）。**独立标签的评分规则及在效用函数上的作用方式需要与互斥标签分开建模**

#### 标签类别_3：绝对实力

> 考虑用户偏向客观维度的能力，在相同维度上评分相似的用户**更可能**被匹配为搭子

1. 参与过的比赛场次
2. 是否获过奖
3. 获奖欲望

# 2		雅思学习搭子

## 2.1		固定标签

1. 对搭子的性别要求
2. 对搭子的年级要求
3. **目标（在这里就是学习搭子）**
4. 对搭子的专业要求**（注意，这个和数学建模大赛是有差异的）**

## 2.2		分类标签

### 2.2.1		互斥标签

#### 标签类别_1：擅长题型（雷达图，相对能力）

1. 听力
2. 阅读
3. 写作
4. 口语

#### 标签类别_2：性格动能因子

> 考虑用户在学习小组中的性格特质与行为倾向。在相同维度上同为评分高的两个用户**更不可能**被匹配为搭子

1. 计划制定及推动者（善于制定学习计划、敦促进度推进）
2. 资源获取者（善于从各个渠道获取学习资源）
3. 协调者（善于组织讨论、推动问题解决）

### 2.2.2		独立标签

> **独立标签**反映一个用户相对独立的指标（和其他标签没关系，不存在互斥标签中"此消彼长"的关系）。**独立标签的评分规则及在效用函数上的作用方式需要与互斥标签分开建模**

#### 标签类别_3：学习目标与投入

> 考虑用户偏向客观维度的学习状态，在相同维度上评分相似的用户**更可能**被匹配为搭子

1. 是否有雅思考试的经历
3. 是否愿意一起参加培训班
4. 每周可投入的共同学习时长（小时，正整数）
5. 目标成绩期望（0~10，越高表示对成绩要求越严格）
